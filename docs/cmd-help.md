> Get Help with Mimic My Voice

You can get help at any time by running:

```bash
mimic help
```

You can get help on a specific command by just adding `help` to the end of the command:

```bash
mimic setup help
mimic studio help
mimic train help
mimic backup help
mimic restore help
```
